# E-commerce Orders Analysis (Case Study)


**Tools**: Python, Pandas, NumPy, Matplotlib, Seaborn

## Overview
This project analyzes an e-commerce dataset to extract insights such as:
- Mean/median of quantities
- Min/max prices
- Revenue per order
- Normalization of features
- Customer and region-level analysis
- Visualizations (line, bar, pie, box plots)
